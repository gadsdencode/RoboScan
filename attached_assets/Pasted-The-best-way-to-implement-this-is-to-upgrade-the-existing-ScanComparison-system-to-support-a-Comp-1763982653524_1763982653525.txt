The best way to implement this is to upgrade the existing `ScanComparison` system to support a "Competitor Mode" (Side-by-Side) alongside the existing "History Mode" (Diff). This requires a new "Competitor Analysis" workflow in the dashboard that can trigger scans on two URLs simultaneously and then present them in a revamped comparison view.

Here is the implementation plan:

1.  **Update `scanComparison.ts`**: Add logic to generate a "Bot Permission Matrix" to easily spot differences in bot access between two sites.
2.  **Revamp `ScanComparison.tsx`**: Transform it into a tabbed interface (Summary, Bot Matrix, File Diff) that handles both "Previous vs Current" and "My Site vs Competitor" contexts.
3.  **Update `dashboard.tsx`**: Add a "Compare Sites" feature that allows users to input two distinct URLs, scans them (if needed), and opens the comparison view.

### 1\. Update Comparison Logic

Add the matrix generation logic to the shared library.

```typescript
// client/src/lib/scanComparison.ts

import type { Scan } from "@shared/schema";

// ... existing interfaces ...

export interface BotPermissionComparison {
  bot: string;
  valA: string;
  valB: string;
  status: 'same' | 'different' | 'unique_to_a' | 'unique_to_b';
}

export function getBotPermissionMatrix(scanA: Scan, scanB: Scan): BotPermissionComparison[] {
  const permsA = (scanA.botPermissions as Record<string, string>) || {};
  const permsB = (scanB.botPermissions as Record<string, string>) || {};
  
  const allBots = Array.from(new Set([...Object.keys(permsA), ...Object.keys(permsB)])).sort();
  
  return allBots.map(bot => {
    const valA = permsA[bot];
    const valB = permsB[bot];
    
    let status: BotPermissionComparison['status'] = 'same';
    
    if (valA !== valB) {
      if (!valA) status = 'unique_to_b';
      else if (!valB) status = 'unique_to_a';
      else status = 'different';
    }
    
    return { 
      bot, 
      valA: valA || '-', 
      valB: valB || '-', 
      status 
    };
  });
}
```
